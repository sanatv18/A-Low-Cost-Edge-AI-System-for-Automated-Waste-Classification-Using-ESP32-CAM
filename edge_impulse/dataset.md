# Dataset

## Dataset Description

The dataset was collected and labeled using Edge Impulse Studio.

### Classes

- Paper
- Battery

### Data Collection

Images were captured using the ESP32-CAM under different conditions to improve model robustness.

The dataset includes:

- Multiple viewing angles
- Different lighting conditions
- Various backgrounds

## Data Split

- Training Set
- Validation Set
- Testing Set

## Deployment

The trained model was exported as an Arduino Library and integrated into the ESP32-CAM firmware for real-time inference.
